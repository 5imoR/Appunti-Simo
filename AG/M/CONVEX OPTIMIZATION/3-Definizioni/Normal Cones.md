## Definition \[Normal Cones]

Dato un convex set $C$ ed un punto $x\in C$  il *normal cone* $N_C(x)$ Ã¨ definito come:
$$
N_C(x)=\set{d\in\mathbb R^n |d^T(y-x)\le 0\quad\forall y\in C}
$$
A parole sono:
- Il set di direzioni perpendicolari a $C$ nel punto $x$
- Il set di vettori che definiscono supporting hyperplanes  nel punto $x$
